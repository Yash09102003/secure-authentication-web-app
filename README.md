# Secure Authentication Web App

## Features
- Email/Password Authentication
- Google OAuth Login
- Forgot Password Reset
- Firestore User Profile Storage
- Responsive Login and Signup UI
- Protected User Dashboard

## Tech Stack
- HTML
- CSS
- JavaScript
- Firebase Authentication
- Cloud Firestore

## What I Implemented
This project demonstrates secure multi-provider authentication using Firebase, user profile persistence with Firestore, and frontend validation for login/signup workflows.

## Future Improvements
- Two-factor authentication
- Role-based access control
- User profile editing
- Session management
